Obtaining Tpetra
################

Tpetra_ is package within Trilinos_ which is maintained on `GitHub
<https://www.github.com>`_.  To obtain a copy of Tpetra, Trilinos can
be downloaded from the `Trilinos GitHub <https://www.github.com/trilinos/Trilinos>`_
site or cloned on the command line:

.. code-block:: sh

   git clone https://www.github.com/trilinos/Trilinos
