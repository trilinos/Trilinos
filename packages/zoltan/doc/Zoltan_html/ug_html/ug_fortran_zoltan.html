FORTRAN: Compiling Zoltan

To include the Fortran interface in the Zoltan library, use the YES_FORTRAN parameter in the make statement; for example
gmake  YES_FORTRAN=1  ZOLTAN_ARCH=<platform>  zoltan
Before compiling the library, make sure that the application's zoltan_user_data.f90 has been placed in the Zoltan/fort/ directory.
