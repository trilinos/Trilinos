Test directories:

correctness/    
  Tests to check correctness of Zoltan2 answers.

data/     
  Data for tests.

helpers/  
  Classes and methods used by tests.

order/      
  Ordering tests.

partition/  
  Partitioning tests.

scaling/  
  Large scale performance tests.

temp/     
  Directory to facilitate short term tests.

unit/
  Unit tests for each Zoltan2 class or namespace method.
